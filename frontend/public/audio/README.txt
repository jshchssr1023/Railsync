SIMPSONS THEME EASTER EGG
=========================

To enable the Ralph Wiggum audio easter egg:

1. Add a short (5-8 second) Simpsons theme audio clip to this folder
2. Name it: simpsons-theme-short.mp3
3. Click the "Ralph Mode" toggle in the dashboard header to enable

The theme will play once per session when the dashboard opens.

NOTE: You must source this audio legally (fair use snippet, public domain, etc.)

"Me fail English? That's unpossible!" - Ralph Wiggum
